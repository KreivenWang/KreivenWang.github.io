说明

box-shadow向框添加一个或多个阴影。该属性是由逗号分隔的阴影列表，每个阴影由 2-4 个长度值、可选的颜色值以及可选的inset关键词来规定。省略长度的值是 0。

值 描述
h-shadow 必需。水平阴影的位置。允许负值。
v-shadow 必需。垂直阴影的位置。允许负值。
blur 可选。模糊距离。
spread 可选。阴影的尺寸。
color 可选。阴影的颜色。请参阅 CSS 颜色值。
inset 可选。将外部阴影 (outset) 改为内部阴影。

EasyUI中DateBox、DateTimeBox的格式问题

发表于 2017-07-04 | 分类于 代码笔记 | 阅读次数

问题描述

以DateBox为例，当属性值的格式日期分隔符为-，如yyyy-MM-dd，而DateBox的默认格式为yyyy/MM/dd时，会导致DateBox无法赋值，此时将显示为当前时间，而不是对应的属性值。

解决办法

引入'easyui/locale/easyui-lang-zh_CN.js'，会把日期控件的默认格式改为'yyyy-MM-dd'。
